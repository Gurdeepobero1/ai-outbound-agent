# AI Outbound Intelligence Agent

An AI-powered tool that generates personalized sales outreach messages based on product, ICP, and lead details.

## Features
- Personalized cold message generation
- Follow-up suggestions
- Tone control
- Fast inference using Groq API

## Tech Stack
- Python
- Streamlit
- Groq API

## Run Locally
pip install -r requirements.txt
streamlit run app.py
